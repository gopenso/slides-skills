# 主題色預設（Themes）

5 套精心調配的主題色板，保證「電子雜誌 × 電子墨水」的美學不垮。**不允許使用者自定義色彩——色彩搭配錯了畫面瞬間變醜**，只從以下預設中挑選。

---

## 使用方法

1. 問使用者選哪套（或基於內容推薦一套）
2. 開啟 `assets/template.html` 的 `<style>` 塊
3. 找到開頭的 `:root{` 塊
4. **整體取代**標有「主題色」註釋的那幾行 `--ink` / `--ink-rgb` / `--paper` / `--paper-rgb` / `--paper-tint` / `--ink-tint`
5. 其他 CSS 都走 `var(--...)`，無需任何其他改動

---

## 🖋 墨水經典 (Monocle 預設)

**適合**：通用分享、商業釋出、科技產品、任何場景都安全的預設選取。
**調性**：純墨黑 + 暖米白，雜誌感最強，Monocle / Apricot / A Book Apart 風。

```css
--ink:#0a0a0b;
--ink-rgb:10,10,11;
--paper:#f1efea;
--paper-rgb:241,239,234;
--paper-tint:#e8e5de;
--ink-tint:#18181a;
```

---

## 🌊 靛藍瓷 (Indigo Porcelain)

**適合**：科技/研究/資料分享、工程師文化、深度內容、技術釋出會。
**調性**：深靛藍 + 瓷白，冷靜、理性、有深度，像學術期刊或藍印花瓷器。

```css
--ink:#0a1f3d;
--ink-rgb:10,31,61;
--paper:#f1f3f5;
--paper-rgb:241,243,245;
--paper-tint:#e4e8ec;
--ink-tint:#152a4a;
```

---

## 🌿 森林墨 (Forest Ink)

**適合**：自然/可持續/文化/非虛構內容、戶外品牌、環保主題。
**調性**：深森林綠 + 象牙，沉穩、有呼吸感，像舊版《國家地理》。

```css
--ink:#1a2e1f;
--ink-rgb:26,46,31;
--paper:#f5f1e8;
--paper-rgb:245,241,232;
--paper-tint:#ece7da;
--ink-tint:#253d2c;
```

---

## 🍂 牛皮紙 (Kraft Paper)

**適合**：懷舊/人文/閱讀/歷史/文學分享、獨立雜誌、手作品牌。
**調性**：深棕 + 暖米，像牛皮信封或老筆記本，溫暖、有年代感。

```css
--ink:#2a1e13;
--ink-rgb:42,30,19;
--paper:#eedfc7;
--paper-rgb:238,223,199;
--paper-tint:#e0d0b6;
--ink-tint:#3a2a1d;
```

---

## 🌙 沙丘 (Dune)

**適合**：藝術/設計/創意/時尚分享、畫廊手冊、審美優先的私人分享會。
**調性**：炭灰 + 沙色，克制、高階、中性，像沙漠黃昏或建築設計圖冊。

```css
--ink:#1f1a14;
--ink-rgb:31,26,20;
--paper:#f0e6d2;
--paper-rgb:240,230,210;
--paper-tint:#e3d7bf;
--ink-tint:#2d2620;
```

## 轉場原則

- **一份 deck 只用一套主題**，不要中途換色
- WebGL shader 的預設主色（鈦金色散 / 銀色流動）適配所有 5 套（經測試可接受）
- `currentColor` 驅動程式的 border / icon 會跟隨 section 的 text color 自動適配，無需額外調整
- 選定主題後，`<title>` 文字和 `chrome` 文案可以強化該主題的語義（例如牛皮紙配「Vol.03 · 秋」這種）

## ❌ 不要做的事

- ❌ **不允許混搭**（例如 ink 取墨水經典的，paper 取沙丘的）——會徹底違和
- ❌ **不允許使用者隨便給一個 hex 值**——需委婉拒絕並展示 5 套預設讓選
- ❌ **不要直接修改 template.html 其他地方的色彩**——所有散落 rgba 都走 var，改 :root 一處即可

選定主題後在 skill 對話中告訴使用者：「用 🖋 墨水經典 / 🌊 靛藍瓷 ...」並在 deck 專案記錄裡備註，方便後續迭代時保持一致。
